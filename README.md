# Web Learning Vocabulary

## Giới thiệu
Website hỗ trợ học từ vựng tiếng Anh.

## Chức năng
- Quản lý từ vựng
- Làm bài Quiz
- Điền từ vào đoạn văn
- Sinh đoạn văn bằng AI

## Công nghệ
- ASP.NET Core Web API
- HTML/CSS/JavaScript
- Gemini API

## Cách chạy
1. Clone project
2. Cấu hình API Key trong appsettings.Development.json hoặc User Secrets
3. Chạy:

dotnet run
